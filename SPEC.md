## 一、核心定位

**产品名**: CodeLens AI
**定位**: 专注AI生成代码的质量保障平台
**标语**: "让AI代码从风险到可信"

### 目标用户
- 使用AI编程工具的开发者（Copilot, Cursor等）
- 需要代码质量保障的中小团队
- 对AI代码安全有要求的企业

---

## 二、核心功能（MVP）

### F1: AI代码分析引擎
- **输入**: 代码片段 + 语言类型
- **输出**: 问题列表 + 严重级别 + 修复建议
- **检测类型**:
  - 逻辑错误（空指针、数组越界、类型不匹配）
  - 安全漏洞（注入风险、不安全依赖、凭证泄露）
  - 技术债务（重复代码、过时API、低可维护性）

### F2: 静态规则引擎
- 内置100+ AI代码专项规则
- 基于AST的代码分析
- 支持: Python, JavaScript, TypeScript, Java, Go

### F3: CI/CD集成
- GitHub Actions插件
- GitLab CI模板
- Webhook实时触发

### F4: 报告与仪表盘
- 问题汇总报告（PDF/HTML）
- 团队问题趋势图
- 个人代码质量分数

## 四、API设计

### POST /api/v1/analyze
分析代码片段

```json
// Request
{
  "code": "def foo():\n    return None",
  "language": "python",
  "options": {
    "check_security": true,
    "check_logic": true,
    "check_debt": true
  }
}

// Response
{
  "issues": [
    {
      "type": "logic",
      "severity": "high",
      "line": 2,
      "message": "函数返回值可能为None，建议添加类型注解",
      "rule_id": "AI-LG-001",
      "fix": "添加类型注解: def foo() -> Optional[Any]:"
    }
  ],
  "summary": {
    "total": 1,
    "high": 1,
    "medium": 0,
    "low": 0
  },
  "score": 85
}
```

### GET /api/v1/health
